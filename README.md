# ai4devs

> The organization landing page for [ai4devs](https://ai4devs.github.io) — a research-driven team building open-source developer tools at the intersection of AI, code quality, and prompt engineering.

Live at **https://ai4devs.github.io**

## Structure

| File | Contents |
|------|----------|
| `index.html` | Landing page (nav, hero, tools, about, team) |
| `assets/style.css` | Shared design system — dark theme, amber/teal accents, Inter |

## Tools

- **[Prompt Radar](https://ai4devs.github.io/prompt-radar)** — a VS Code extension that analyzes prompts-as-code and scores them across Security, Safety, Formatting, Efficiency, and Reliability.

## Adding a tool

Duplicate the `.tool-card` article in `index.html` under the Tools section and update its title, tag, description, and links.
